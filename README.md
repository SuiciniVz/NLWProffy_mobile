# Getting Started

```bash
# Clone Repository
$ git clone https://github.com/SuiciniVz/NLWProffy_mobile.git

# Install Dependencies
$ yarn install

# Run Aplication
$ yarn start
```
:eyes: Read the QR Code with the app of expo or run on emulator.
